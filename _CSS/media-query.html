css 미디어 쿼리

css 미디어 쿼리란?

@media 태그를 사용하여 사용자가 사용하는 기기의 너비와 높이, 가로 모드 또는 세로 모드 등을 감지하여 조건에 해당할 경우에만 스타일을 적용해주는 CSS3의 속성이다.

css 미디어 쿼리 구문

@media not|only media-type and (media-feature and|or|not media-feature) {
CSS-Code;
}

css 미디어 쿼리 유형(media-type)

@media all - 전체

@media screen - 컴퓨터, 태블릿, 모바일 등 화면일 경우에 적용

@media print - 사용자가 프린트할 경우에 적용

@media speach - 페이지를 소리내어 "읽는" 스크린리더에 적용

css 미디어 쿼리 기능(media-feature)

너비와 높이 = max-width, min-width, max-height, min-height

뷰포트 방향 = orientation (landscape[가로], portrait[세로])
