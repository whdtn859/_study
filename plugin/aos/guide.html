주의사항

AOS 라이브러리 애니메이션은 블록 태그(Block Tag)에만 적용 가능합니다.

이 라이브러리를 사용한 요소는 transform: none; 속성이 강제되기 때문에 다른 transform을 사용하는데 제약이 있습니다.



yarn, npm, bower 다운 방법

yarn add aos

npm install aos --save

bower install aos --save



AOS 사용방법

  • 애니메이션 효과를 사용할 요소에 data-aos 속성을 추가하면 작동 됩니다.
  • ex ) <div data-aos="fade-up"></div>
  • 옵션을 개별적으로 줄 수 있습니다.
  • ex ) <div data-aos="fade-up" data-aos-duration="1500" data-aos-easing="ease-in-sine"></div>
  • 스크립트로 옵션 작성 시 공통적으로 적용할 수 있습니다.
